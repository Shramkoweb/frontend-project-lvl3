# Описание проекта
RSS-лента новостей с автоматическим обновлением.

## Технологии
- Браузерная инфраструктура (webpack)
- MVC
- Jquery
- i18next
- Bootstrap
- AJAX используя Axios
- События, манипуляции с DOM
- Валидация, DOMParser
- Деплой (https://vercel.com)
